# 📧 AI 智能邮件助手

一个基于 DeepSeek API 和 Streamlit 的 AI 邮件助手，可自动润色邮件、记录与不同公司的历史对话，并一键发送真实邮件。

## ✨ 功能特性

- 🤖 AI 智能润色：输入大白话，自动生成正式商务邮件
- 🧠 多公司记忆：为每家公司独立保存对话历史
- 📤 一键发送：集成 QQ 邮箱 SMTP，发送真实邮件
- 📂 历史查看：在 Web 端查看与任意公司的往来记录
- 💾 本地保存：支持将邮件保存到本地文件
